Site Web GSB

Développement d'un site web pour la gestion des médicaments et des activités de GSB

Ce projet consiste à créer une plateforme web pour la gestion des médicaments fabriqués par GSB et des activités proposées par l'entreprise. Le site permet aux utilisateurs de consulter des informations détaillées sur les médicaments, leurs effets thérapeutiques, leurs effets secondaires et leurs interactions avec d'autres médicaments. De plus, le site propose des activités complémentaires, avec la possibilité de s'inscrire à ces activités.

Technologies utilisées

  • PHP - Langage de programmation principal utilisé pour la logique métier et le back-end du site.
  • MVC - Architecture du projet, divisée en trois parties : Modèle, Vue et Contrôleur, permettant une gestion claire et évolutive du projet.
  • API REST - Utilisée pour l'accès à la base de données, permettant une gestion efficace des données côté serveur.
  • Bootstrap - Framework CSS utilisé pour rendre le site responsive et agréable visuellement sur différents appareils.
  • GitHub - Plateforme utilisée pour la gestion de version du code, avec un travail en binôme et une méthodologie agile.

Fonctionnalités principales

  • Présentation des médicaments fabriqués par GSB, avec détails sur les effets thérapeutiques, secondaires et les interactions avec d'autres médicaments.
  • Inscription aux activités complémentaires proposées par GSB.
  • Mentions légales du site web, respectant les obligations juridiques et les droits d'image.

Procédé de développement

Le projet a été réalisé en suivant la méthode agile, avec un travail en binôme. Le processus de développement a été structuré en sprints, permettant d'avancer de manière progressive tout en ajustant les fonctionnalités et l'interface utilisateur au fur et à mesure. GitHub a été utilisé pour la gestion de version et le suivi de l'avancement du projet.
