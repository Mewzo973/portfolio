Projet Android

Ce projet Android consiste à créer une application mobile permettant aux utilisateurs de gérer leurs tâches quotidiennes de manière efficace et intuitive.

Détails Techniques

Cette application a été développée en utilisant Android Studio et le langage Kotlin. Voici quelques-unes des fonctionnalités clés :

  • Création, modification et suppression de tâches.
  • Notifications pour rappeler les tâches à accomplir.
  • Interface utilisateur fluide et réactive.
  • Base de données locale pour stocker les tâches.
