Gestion des RDV

Application de gestion des RDV pour les visiteurs médicaux

Cette application, réalisée avec Android Studio et une base de données SQLite embarquée, permet de gérer les rendez-vous des professionnels médicaux. Elle inclut plusieurs fonctionnalités importantes comme l'enregistrement des professionnels, la prise de RDV, l'affichage des plannings, et la recherche de professionnels par ville ou code postal.

Technologies utilisées

  • Android Studio
  • SQLite (base de données embarquée)
  • Java et XML pour l'interface et la logique

Fonctionnalités principales

  • Enregistrement d'un professionnel médical (nom, prénom, spécialité, adresse, contact)
  • Prise de rendez-vous (choix de la date, de l'heure et du professionnel)
  • Affichage du planning d'une journée
  • Recherche de professionnels par ville ou code postal
